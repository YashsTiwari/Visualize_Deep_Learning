# Visualize_Deep_Learning
This repo contains visualizations for multiple deep learning concepts. I hope these will help you to understand the concepts better.


### How to use this repo?

- Either do git clone of this repo or download the ZIP file by clicking on the "Code" button in green, then a drop down will appear with the option of "Download ZIP".
- Extract the ZIP in case you have downloaded the ZIP file.
- Just double click on the file and it will open in web browser and you can access the animations.
- (Optional) In case code file is opening on double clicking, then right click on the file, press the option which says "open with", then choose any browser. This will open the animation in the web browser.

Happy learning! :)
