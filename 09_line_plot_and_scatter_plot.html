📊 Data Visualization Intuition
📈 Line Plot
Line Plot = Connected Data Points
Days
Temperature
A line plot helps us see change over time
✅ Use line plots when:

• data changes over time
• trends are important
• we want to see increase/decrease

Examples:
🌡 Temperature
📈 Stock prices
📉 Training loss in Deep Learning
days = [1,2,3,4]
temp = [20,25,22,30]

plt.plot(days,temp)
🎯 Scatter Plot
Scatter plots help visualize different groups/classes
Weight
Height
Apples
Oranges
Scatter plots show relationships between data points
✅ Use scatter plots when:

• visualizing data points
• comparing groups/classes
• understanding clustering
• seeing separation between categories

This becomes VERY important in:
🤖 Classification
🧠 Neural Networks
🎯 Decision Boundaries
plt.scatter(x,y)
